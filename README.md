# 2to1screen

**Качественные скриншоты одного монитора на Windows 11 с двумя экранами.**

Когда к ПК подключены два монитора, стандартное сочетание **Win + Print Screen** делает снимок сразу обоих экранов и сильно режет качество. **2to1screen** перехватывает клавишу Print Screen и сохраняет снимок **только выбранного монитора** в полном качестве — в буфер обмена и в файл.

Приложение оформлено в стиле серебристого **Liquid Glass**: полупрозрачный фон, сквозь который видно обои рабочего стола. В версии 1.1 стекло стало ещё прозрачнее, появились выбор цвета темы, отдельный **Магазин тем** с анимированными оформлениями и настраиваемые глобальные горячие клавиши.

---

## Что нового в 1.1

- 🫧 **Более прозрачный Liquid Glass** — сильнее просвечивают обои, текст при этом остаётся читаемым (ярче и с лёгкой тенью).
- 🎨 **Цвет темы Liquid Glass** — палитра рядом с переключателем: выбираете основной цвет, и он применяется ко всем стеклянным элементам (акценты, блики, прозрачность). Есть и «Свой цвет…».
- 🖼️ **Магазин тем** — отдельный раздел с современными оформлениями (Midnight, Graphite, Aurora, Synthwave, Ocean Deep, Ember, Evergreen, Pearl), часть из них **анимированные**. Клик колёсиком мыши по теме — полноэкранный предпросмотр. При включённом Liquid Glass выбор темы спросит подтверждение (красная «Выключить Liquid Glass» / серая «Отмена»).
- ⌨️ **Настраиваемые горячие клавиши** — клавишу снимка можно назначить на любую комбинацию (например, Win + F12). Отдельная глобальная комбинация ставит перехват на паузу и снимает её **даже когда окно свёрнуто в трей**.
- 🔊 **Починен звук затвора** — теперь воспроизводится встроенный звук снимка независимо от схемы звуков Windows (есть кнопка «Проверить»).

## Возможности

- 🖥️ **Снимок одного монитора** — выбираете нужный экран, Print Screen снимает только его.
- ✨ **Liquid Glass** — прозрачная стеклянная тема с выбором основного цвета (можно полностью отключить).
- 🎨 **Магазин тем** — современные и анимированные оформления с полноэкранным предпросмотром.
- ⌨️ **Глобальные горячие клавиши** — своя комбинация для снимка и для паузы перехвата (работает из трея).
- ⚡ **Запуск одной кнопкой** — кнопка «Запустить» включает перехват Print Screen.
- 🛠️ **Автозапуск в фоне** — старт вместе с Windows с правами администратора (через UAC).
- 🎚️ **Качество снимка** — Очень высокое / Высокое / Среднее / Низкое или **своё** разрешение и формат (PNG / JPEG).
- 📋 **Гибкое сохранение** — буфер обмена и/или файл, выбор папки, звук затвора, уведомления.
- 👤 **Разработчик** — [vk.ru/dirroladd](https://vk.ru/dirroladd)

## Требования

- **Только Windows 11** (сборка 22000 и выше). На более старых версиях приложение покажет предупреждение и не запустится.
- Архитектура x64.
- Среда выполнения не нужна — `.exe` самодостаточный (self-contained).

## Установка

1. Скачайте `2to1screen.exe` со страницы [Releases](../../releases/latest).
2. Запустите файл (Windows SmartScreen может предупредить о неизвестном издателе — «Подробнее» → «Выполнить в любом случае»).
3. На вкладке **Приложение** нажмите **Запустить**.
4. Нажмите **Print Screen** — снимок выбранного монитора попадёт в буфер обмена и в папку сохранения.

## Использование

- **Приложение** — кнопка «Запустить/Остановить», переключатели «Запускать автоматически» и «Выключить Liquid Glass», палитра цвета Liquid Glass.
- **Настройки** — выбор монитора, качество скриншота (включая своё), горячие клавиши, папка и параметры сохранения.
- **Магазин тем** — выбор и предпросмотр готовых тем (доступен при выключённом Liquid Glass).
- **Разработчик** — ссылка на автора.
