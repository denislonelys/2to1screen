# 2to1screen

**Качественные скриншоты одного монитора на Windows 11 с двумя экранами.**

Когда к ПК подключены два монитора, стандартное **Win + Print Screen** делает снимок сразу обоих экранов и сильно режет качество. **2to1screen** перехватывает Print Screen и сохраняет снимок **только выбранного монитора** в полном качестве — в буфер обмена и в файл.

---

## Возможности

- 🖥️ **Снимок одного монитора** — выбираете нужный экран, Print Screen снимает только его.
- 🎨 **Магазин тем (500+)** — большая библиотека тем; превью прямо в приложении.
- 🧩 **Свои темы** — лёгкая (ссылка на PNG + цвет акцента) или продвинутая (вставить код темы).
- ✨ **Liquid Glass** — прозрачная стеклянная тема с настраиваемой прозрачностью (по умолчанию выключена, стоит чёрная тема).
- 🔑 **Активация по ключу + HWID** — пробный период 2 дня, затем активация ключом, привязанным к устройству.
- ♻️ **Автообновление** — программа сама проверяет версию при запуске и обновляется.
- 🪟 **Win + D на один монитор** — сворачивает окна только на мониторе под курсором.
- ⌨️ **Горячие клавиши** — назначаются на любое действие (в т.ч. кликом колёсиком по элементу).
- 🛠️ **Автозапуск в фоне** — старт вместе с Windows с правами администратора (через UAC).
- 🎚️ **Качество снимка** — Очень высокое / Высокое / Среднее / Низкое или **своё** разрешение и формат (PNG / JPEG).

## Требования

- **Только Windows 11** (сборка 22000 и выше). На более старых версиях приложение покажет предупреждение и не запустится.
- Архитектура x64. Среда выполнения не нужна — `.exe` самодостаточный (self-contained).

## Установка

1. Скачайте программу: **https://orproject.ru/2to1screen.exe**
2. Запустите файл (Windows SmartScreen может предупредить — «Подробнее» → «Выполнить в любом случае»).
3. При первом запуске введите ключ активации или нажмите **«Попробовать бесплатно»** (2 суток).
4. На вкладке **Приложение** нажмите **Запустить** и пользуйтесь Print Screen.

## Активация и ключи

- Каждому устройству выдаётся уникальный **HWID** (на основе оборудования — не сбрасывается переустановкой Windows).
- **Пробный период** — 2 суток, один раз на устройство. В это время доступны только запуск и фоновый режим, в углу экрана висит счётчик «не активировано».
- **Ключи** выдаёт администратор через Telegram-бота. Бот умеет генерировать, продлевать и обнулять ключи, а также блокировать/разблокировать HWID.
