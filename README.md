# 2to1screen

**Качественные скриншоты одного монитора на Windows 11 с двумя экранами.**

Когда к ПК подключены два монитора, стандартное сочетание **Win + Print Screen** делает снимок сразу обоих экранов и сильно режет качество. **2to1screen** перехватывает клавишу Print Screen и сохраняет снимок **только выбранного монитора** в полном качестве — в буфер обмена и в файл.

Приложение оформлено в стиле серебристого **Liquid Glass**: полупрозрачный фон, сквозь который видно обои рабочего стола.

---

## Возможности

- 🖥️ **Снимок одного монитора** — выбираете нужный экран, Print Screen снимает только его.
- ✨ **Liquid Glass** — прозрачная стеклянная тема (можно полностью отключить).
- ⚡ **Запуск одной кнопкой** — кнопка «Запустить» включает перехват Print Screen.
- 🛠️ **Автозапуск в фоне** — старт вместе с Windows с правами администратора (через UAC).
- 🎚️ **Качество снимка** — Очень высокое / Высокое / Среднее / Низкое или **своё** разрешение и формат (PNG / JPEG).
- 📋 **Гибкое сохранение** — буфер обмена и/или файл, выбор папки, звук затвора, уведомления.
- 👤 **Разработчик** — [vk.ru/dirroladd](https://vk.ru/dirroladd)

## Требования

- **Только Windows 11** (сборка 22000 и выше). На более старых версиях приложение покажет предупреждение и не запустится.
- Архитектура x64.
- Среда выполнения не нужна — `.exe` самодостаточный (self-contained).

## Установка

1. Скачайте `2to1screen.exe` со страницы [Releases](../../releases/latest).
2. Запустите файл (Windows SmartScreen может предупредить о неизвестном издателе — «Подробнее» → «Выполнить в любом случае»).
3. На вкладке **Приложение** нажмите **Запустить**.
4. Нажмите **Print Screen** — снимок выбранного монитора попадёт в буфер обмена и в папку сохранения.

## Использование

- **Приложение** — кнопка «Запустить/Остановить», переключатели «Запускать автоматически» и «Выключить Liquid Glass».
- **Настройки** — выбор монитора, качество скриншота (включая своё), папка и параметры сохранения.
- **Разработчик** — ссылка на автора.

```bash
  -p:PublishSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true \
  -p:EnableCompressionInSingleFile=true -o publish
```
